# Python and Libraries 🐍

A beginner-to-intermediate friendly repository covering **Python fundamentals** and **commonly used Python libraries**. This repo is designed to build a strong foundation for **software development, data science, AI, and machine learning**.

---

## 📌 Purpose

This repository helps you:

* Learn core Python concepts step by step
* Understand and use popular Python libraries
* Practice with examples and small scripts
* Prepare a base for AI / ML / Data Science projects

---

## 🧠 Topics Covered

### 🔹 Python Core

* Variables & Data Types
* Conditional Statements
* Loops
* Functions
* Modules & Packages
* File Handling
* Exception Handling
* OOP Concepts
* Advance Python Concepts

### 🔹 Libraries

* **NumPy** – numerical computing
* **Pandas** – data manipulation
* **Matplotlib** – data visualization
* **Seaborn** – statistical visualization

(More libraries will be added over time)

---

## 🛠 Requirements

* Python 3.8+
* pip

Install required libraries:

```bash
pip install numpy pandas matplotlib seaborn
```

---

## 🚀 How to Use

1. Clone the repository

```bash
git clone https://github.com/your-username/python-and-libraries.git
```

2. Navigate to the folder

```bash
cd python-and-libraries
```

3. Run any Python file

```bash
python filename.py
```
